💡 解决 fullpage.js 的未授权报错问题

由于 fullpage.js 最新版本修改了开源协议，需要购买授权后才可以使用，否则控制台会出现Fullpage.js version 3 has changed its license to GPLv3...报错信息。不过通过简单修改源码，可以清除掉控制台报错，并正常使用插件。

如果使用的是普通的 fullpage.js，打开未压缩的源文件搜索下列代码删除即可：

if(!isOK){
    showError('error', 'Fullpage.js version 3 has changed its license to GPLv3 and it requires a `licenseKey` option. Read about it here:');
    showError('error', 'https://github.com/alvarotrigo/fullPage.js#options.');
}

如果使用的是Vue-fullpage.js，则需要找到node_modules/vue-fullpage.js/dist/vue-fullpage.js文件，在其中搜索fullPage:字符串，将其所在函数的代码删除，但不要删除函数本身。

function Vn(e,t){rn.console&&rn.console[e]&&rn.console[e]("fullPage: "+t)}

function Vn(e,t){}
