🔔 JavaScript 中的定时器

定时器可以用来设置某段代码在特定时间之后 / 每隔特定时间执行。

创建定时器

通过setTimeout()和setInterval()两个方法可以创建定时器。前者设置代码在特定时间后执行，只执行一次；后者设置代码每隔特定时间循环执行，会执行多次。

它们的第一个参数为回调函数，第二个参数为定时器的时间，单位为毫秒。

const id1 = setTimeout(() => {
    console.log('延迟一秒后输出！')
}, 1000)

const id2 = setInterval(() => {
    console.log('每隔一秒输出！')
}, 1000)

如果传入三个以上的参数，则之后的参数表示回调函数的参数：

// 1 秒后输出 hello
setTimeout(msg => { 
    console.log(msg)
}, 1000, 'hello')

清除定时器

创建定时器之后，这两个方法会返回一个id，将id传入对应的clearTimeout()和clearInterval()可以清除定时器：

clearTimeout(id1)
clearInterval(id2)
